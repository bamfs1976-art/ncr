# NCR Fantasy Planner

Nations Championship 2026 fantasy planner for the July rounds, with a match predictor, Monte Carlo season simulator and Grand Final picker.

- **Fantasy planner** — rules-checked 16-player squad builder inside 100M, with fixture-led recommendations per round.
- **Predictor** — enter scores and tries for all 36 fixtures; conference standings update live using World Rugby bonus points.
- **Simulator** — Monte Carlo season projection (1,000–10,000 runs) giving conference and title odds. Entered scores are locked into every run.
- **Grand Final** — pick your 2026 Nations Champion.

Single file, vanilla HTML/CSS/JS, no dependencies, no network calls. Predictions save to your browser's local storage.

Prediction features adapted from [MatchMetric/Nations-Championship-2026](https://github.com/MatchMetric/Nations-Championship-2026) and the simulation approach from [ThevinRahu/SportsMetrics](https://github.com/ThevinRahu/SportsMetrics).
